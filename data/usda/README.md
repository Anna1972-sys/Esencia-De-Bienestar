# USDA FoodData Central

Coloca aquí exportaciones oficiales de USDA FoodData Central cuando quieras incorporarlas a la tabla unificada `food_items`.

Nombre recomendado:

```text
usda_fooddata_export.json
```

También puedes usar CSV si contiene columnas normalizadas de nutrientes por 100 g.

No se descarga nada automáticamente.
